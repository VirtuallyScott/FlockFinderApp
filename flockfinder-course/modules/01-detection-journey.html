The Secret Life of a Detection

What FlockFinder actually does — and the journey your data takes in the half-second after a camera is found.

What Is This App, Really?

Imagine you're driving around your neighborhood with a small electronic box plugged into your car. That box is constantly sniffing the air for Wi-Fi signals, looking for patterns that match known surveillance cameras — particularly Flock Safety cameras, which are ALPR cameras used to photograph license plates.

When that box finds one, it beams a message to your iPhone over Bluetooth. Your phone lights up, plays an alert, records the GPS coordinates, and saves everything to a private database that only you can see.

📡

The Hardware

A ESP32-S3 FeatherS3 device scans Wi-Fi signals and broadcasts detections over BLE.

📱

The App

This iOS app is the companion that receives detections, logs GPS location, and stores everything locally.

🗺️

The Map

Every detection appears as a point on your personal map — building a local picture of surveillance infrastructure.

The Orchestration Code

Every time the BLE hardware sends a detection, this function runs. It's the conductor — reading from one source and directing all the others.

CODE 

private func handleDetection(_ bleData: BLEManager.DetectionData) {
    let locationData = locationManager.getLocationData()
    let detection = FlockDetection(bleData: bleData, location: locationData)
    guard detection.confidence >= appSettings.minimumConfidence else { return }
    databaseManager.insertDetection(detection)
    if appSettings.audibleAlertsEnabled {
        audioManager.playDetectionAlert()
    }
}
PLAIN ENGLISH

When a detection arrives from Bluetooth, run this function...

Ask the GPS system for the current location right now.

Combine the camera data with the GPS data into one complete record.

If the detection's confidence score is too low, ignore it and stop here.

Save the full record to the database.

If the user has sound alerts turned on...

...play the detection sound.

Close the if-block.

End of function.

💡
The Coordinator Pattern

Notice how handleDetection doesn't do any of the work itself — it just calls other managers. This is a famous software pattern called the Coordinator pattern. When you ask an AI to "add a feature when a detection happens," this is the exact function to tell it to modify.

What Gets Saved for Every Detection

A single detection is far more than just "camera found here." It's a rich snapshot of your environment at that exact moment.

📸
Camera Identity

Device type (Flock Safety, Ring, Nest…), MAC address, and Wi-Fi SSID.

📍
Location

Latitude, longitude, altitude, and GPS accuracy — precise enough to place a map pin within a few meters.

🚗
Motion Context

Speed in mph, compass heading, and activity type (walking, driving, cycling) from the motion sensor.

📊
Signal Strength & Confidence

RSSI in dBm and a confidence score (0–1) estimating how certain the detection is.
